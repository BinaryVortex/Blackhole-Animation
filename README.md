# Blackhole Animation

Blackhole Animation — a visually pleasing black hole / accretion disk animation built with HTML, CSS and JavaScript.

![Blackhole Animation screenshot](https://github.com/BinaryVortex/Blackhole-Animation/blob/main/Screenshot%202024-07-11%20043133.png?raw=true)

## Description
A lightweight, single-page animation that simulates a black hole and surrounding accretion disk using web technologies. It's a small, portable demo suitable for learning or embedding in a portfolio.

## Features
- Smooth, GPU-friendly animation in the browser
- Single-file demo (index.html + script.js + styles)
- Easy to run locally and modify

## View / Run locally
1. Clone the repository:
   ```
   git clone https://github.com/BinaryVortex/Blackhole-Animation.git
   cd Blackhole-Animation
   ```

2. Open the demo:
   - Double-click `index.html` to open it in your browser, or
   - Serve it locally (recommended for best results):
     ```
     # Python 3
     python3 -m http.server 8000
     # then visit http://localhost:8000 in your browser
     ```

## Files of interest
- `index.html` — entry point and demo page
- `script.js` — animation logic (JavaScript)
- `Screenshot 2024-07-11 043133.png` — demo screenshot used above

## Customization
- Tweak parameters and visual behavior in `script.js` and the CSS in `index.html` to change particle counts, speeds, colors, and other effects.
- The project is intentionally small to make experimenting easy.
